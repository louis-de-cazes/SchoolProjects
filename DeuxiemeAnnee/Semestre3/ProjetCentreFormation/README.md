## 🇫🇷 Explication et Analyse

### 1. Explication

Dans ce projet tutoré, nous avons choisi de réaliser un rapport présentant le rendement économique des centres de formation à travers la vente des joueurs qui y sont formés. Ce sujet nous a semblé particulièrement intéressant pour comprendre comment les clubs de football parviennent à rentabiliser leurs structures de formation, et ainsi mieux appréhender l'importance de ces dernières dans le football professionnel.

Pour ce faire, nous avons collecté automatiquement les données de tous les joueurs formés au sein des clubs professionnels (première et deuxième divisions) du top 8 européen (Angleterre, France, Allemagne, Espagne, Italie, Pays-Bas, Portugal, Belgique) sur les vingt dernières années. Nous avons également récupéré les informations relatives à leurs transferts (prix de vente, date de la transaction, club acheteur, etc.). Par la suite, nous avons analysé ces données afin d'évaluer la rentabilité économique des centres de formation et d'identifier les facteurs qui l'influencent.

Cette collecte automatisée a été réalisée via l'API de Transfermarkt, une base de données de référence sur la carrière des joueurs et les transferts. Nous avons utilisé Python pour formuler les requêtes à l'API ainsi que pour traiter et analyser les données recueillies.

### 2. Analyse

Ce rapport final s'avère particulièrement complet. Malgré un calendrier restreint, nous avons réussi à fournir un travail de qualité qui nous a permis d'obtenir une très bonne note. Ce projet nous a beaucoup appris sur les mécanismes des centres de formation et leur rôle central dans l'écosystème du football professionnel. L'analyse des données a également mis en évidence des tendances intéressantes — notamment concernant les clubs italiens — et a révélé que certains clubs, dont nous n'aurions pas soupçonné un tel niveau de performance, affichent des rendements économiques extrêmement élevés grâce à leur formation.


## 🇬🇧 Explanation and Analysis

### 1. Explanation

As part of this tutored project, which built directly upon our first-year work, we chose to produce a report analyzing the economic return on investment of youth academies through the sale of homegrown players. We selected this topic because we found it compelling to understand how football clubs generate returns from their academy structures, thereby gaining a deeper insight into the significance of these academies within professional football.

To achieve this, we automatically collected data on all players trained in the academies of professional clubs (first and second divisions) across the top 8 European leagues (England, France, Germany, Spain, Italy, Netherlands, Portugal, Belgium) over the last twenty years. We also gathered details regarding their transfers (sale price, transfer date, buying club, etc.). Subsequently, we analyzed this data to understand the economic efficiency of youth academies and the key factors influencing it.

This automated data collection was carried out using the Transfermarkt API, a comprehensive database for player careers and transfers. We used Python to handle the API requests and to process and analyze the gathered dataset.

### 2. Analysis

We believe this final report is highly comprehensive. Despite the limited timeframe, we managed to deliver quality work that earned us an excellent grade. This project taught us a great deal about youth academies and their pivotal role in the professional football industry. Furthermore, we uncovered interesting patterns in the data—particularly regarding Italian clubs—and identified several clubs achieving remarkably high economic returns through their academies, far exceeding our initial expectations.

### 1. Uitleg

In het kader van dit project, dat in het verlengde lag van ons project uit het eerste jaar, hebben we ervoor gekozen een rapport op te stellen over het economisch rendement van jeugdopleidingen op basis van de verkoop van zelfopgeleide spelers. Vervolgens hebben we deze data geanalyseerd om het economisch rendement van de jeugdopleidingen en de factoren die dit rendement beïnvloeden in kaart te brengen.

Deze automatische gegevensverzameling werd uitgevoerd via de API van Transfermarkt, een zeer uitgebreide database voor spelerscarrières en transfers. We hebben Python gebruikt om de API-aanvragen te doen en om de verzamelde gegevens te verwerken en te analyseren.
