## 🇫🇷 Explications et analyse

### 1. Explication

Dans le cadre de notre apprentissage de PHP, nous avons décidé de réaliser un projet de gestion de tournois de football, au sein duquel il était possible, par exemple, d'ajouter des joueurs, des équipes et des matchs. L'application intégrait un système de rôles, distinguant notamment l'administrateur du tournoi de l'entraîneur. Le classement général se mettait à jour automatiquement à l'issue de chaque rencontre, et une page dédiée regroupait les statistiques des joueurs et des équipes. Pour ce faire, nous avons utilisé une base de données MySQL afin de stocker l'ensemble des informations, tandis que l'interface utilisateur a été développée en HTML, CSS et PHP.

### 2. Analyse

Ce projet nous a permis de mettre en pratique nos compétences en PHP et en gestion de bases de données, tout en concevant une application web pleinement fonctionnelle. Nous avons appris à structurer notre code de manière efficace, à gérer les interactions avec la base de données et à créer une interface utilisateur intuitive. De plus, ce travail nous a offert un excellent aperçu des défis inhérents au développement d'une application web complète, notamment en matière de sécurité, de performance et d'expérience utilisateur.

## 🇬🇧 Explanations and analysis

### 1. Explanation

As part of our PHP learning process, we decided to develop a football tournament management project. The platform allowed users to add players, teams, and matches. It featured a role-based access control system, distinguishing for instance between a tournament administrator and a coach. The standings table was automatically updated after each match, and a dedicated page displayed statistics for both players and teams. We utilized a MySQL database to store all the information, and developed the front-end user interface using HTML, CSS, and PHP.

### 2. Analysis

This project enabled us to apply our PHP and database management skills in a practical way while building a fully functional web application. We learned how to structure our code efficiently, handle database interactions, and design an intuitive user interface. Furthermore, this project gave us valuable insight into the challenges of developing a complete web application, particularly regarding security, performance, and user experience.

## 🇳🇱 Uitleg en analyse

### 1. Toelichting

In het kader van onze PHP-opleiding hebben we ervoor gekozen om een project te realiseren voor het beheer van voetbaltoernooien. Binnen dit platform was het bijvoorbeeld mogelijk om spelers, teams en wedstrijden toe te voegen. Het systeem beschikte over een rolgebaseerde toegang, met een onderscheid tussen bijvoorbeeld een toernooibeheerder en een trainer. De ranglijst werd na elke wedstrijd automatisch bijgewerkt en er was een specifieke pagina voor de statistieken van spelers en teams. We hebben een MySQL-database gebruikt om alle informatie op te slaan, en de gebruikersinterface is ontwikkeld met HTML, CSS en PHP.

### 2. Analyse

Dit project heeft ons de kans gegeven om onze vaardigheden in PHP en databasebeheer in de praktijk te brengen, terwijl we een volledig functionele webapplicatie bouwden. We hebben geleerd hoe we onze code efficiënt kunnen structureren, hoe we de interactie met de database moeten beheren en hoe we een intuïtieve gebruikersinterface ontwerpen. Daarnaast heeft dit project ons een goed inzicht gegeven in de uitdagingen die komen kijken bij de ontwikkeling van een complete webapplicatie, met name op het gebied van beveiliging, prestaties en gebruikerservaring.
